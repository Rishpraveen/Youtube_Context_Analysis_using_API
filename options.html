API Configuration

YouTube API

Without an API key, transcripts and comments are extracted directly from the YouTube page. An API key provides more reliable comment fetching.
How to get a YouTube API Key
  1. Go to the Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the YouTube Data API v3
  4. Create credentials (API key)
  5. Copy and paste the API key here

Hugging Face Settings

How to get a Hugging Face API Token
  1. Go to Hugging Face Tokens
  2. Create an account or log in
  3. Create a new token
  4. Copy and paste the token here
  5. Note: Many models can be used without a token

Google Gemini Settings

How to get a Google Gemini API Key
  1. Go to Google AI Studio
  2. Create an account or log in
  3. Create an API key
  4. Copy and paste the key here
  5. Note: Gemini offers a free tier with usage limits
  6. Important: If one model fails, try Gemini 2.0 Flash or 1.5 Flash

Ollama Settings (Local)

How to set up Ollama locally
  1. Install Ollama on your computer
  2. Run Ollama and download a model with: ollama pull llama3.2:3b
  3. Ollama will run locally on port 11434
  4. If you installed models using Alpaca, use the same model name shown by ollama list
  5. No API key needed - completely free and private!
